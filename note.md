zoomで二つのデバイスを同時に使うとハウります.片方の音声を完全に消すことができなくなったせいなのですが,どうすればいいですか?
28 minutes ago

2つのデバイスを同じ部屋でZoomに接続すると、片方をミュートにしてもスピーカーから出る音をもう一方のマイクが拾ってしまうためハウリングが発生します。 
これを防ぐには、単なる「ミュート」ではなく「オーディオの切断（退出）」を行い、そのデバイスのスピーカーとマイクの両方を無効化する必要があります。 
対処方法
パソコンの場合
画面左下の「マイク」アイコンの横にある「＾」をクリックします。
メニューから「コンピューターオーディオから退出」を選択してください。
アイコンがヘッドフォンに「！」がついたマーク（オーディオに未接続の状態）になれば成功です。
スマホ・タブレット（iOS/Android）の場合
画面をタップしてメニューを表示し、「詳細（…）」を選択します。
「オーディオの切断」をタップしてください。
アイコンが「オーディオに接続」というヘッドフォンマークに変われば、音が出なくなりハウリングも止まります。
入室時に設定する場合
入室時のポップアップで「インターネットを使用した通話（Wifiまたは携帯のデータ）」を選択せずに「キャンセル」を押すことで、最初からオーディオ未接続の状態で参加できます。 
どちらのデバイスを「オーディオ切断」にするかお決まりですか？メインで話す方を残し、もう一方は完全に切断するのがおすすめです。

エラーの原因は、pyproject.toml ファイルの [project] セクションに必須項目である name（プロジェクト名） が記述されていないことです。 
pyproject.toml を使ってインストール（pip install .）を行う場合、以下の2つの項目は必須です。 
修正方法
プロジェクトのルートディレクトリにある pyproject.toml を開き、[project] セクションに name と version を追記してください。 
toml
[project]
name = "ariyantype"  # ここにパッケージ名を記述（必須）
version = "0.1.0"    # バージョンも記述（必須）
# ... その他の設定
Use code with caution.

注意点
名前のルール: name に使えるのは、ASCII文字、数字、アンダースコア _、ハイフン -、ピリオド . のみです。
Poetryを使用している場合: もしパッケージ管理に Poetry を使っているなら、[project] ではなく [tool.poetry] セクションに name が書かれている必要があります。ただし、最近のツール環境では標準の [project] セクションに書くことが推奨されています。 
修正後、再度ターミナルで pip install . を実行してみてください。
